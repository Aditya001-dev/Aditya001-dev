## 🎨 Portfolio Technical Deep-Dive

This single HTML file is an engineering showcase in itself. Here's what's under the hood:

<details>
<summary><b>🖼️ Hero Section — WebGL Particle Canvas</b></summary>

A live particle field rendered via `<canvas>` — particles react to mouse movement with spring physics, creating a magnetic field effect. Built in vanilla JS with no libraries, 60fps on all modern devices.

</details>

<details>
<summary><b>📜 Scroll Story — Sticky Scroll Narrative</b></summary>

A cinematic scroll-locked storytelling section driven by a `requestAnimationFrame` loop reading `getBoundingClientRect()` every frame. The key fix: removing `overflow-x: hidden` from `html`/`body` (which silently breaks `position: sticky`) and replacing with `overflow-x: clip` on a page wrapper — the only CSS value that clips without creating a scroll context.

```js
// The math that drives slide transitions
const scrollableRange = container.offsetHeight - vh;
const scrolled        = Math.max(0, Math.min(scrollableRange, -rect.top));
const progress        = scrolled / scrollableRange; // 0 → 1 cleanly
const idx             = Math.min(Math.floor(progress * TOTAL), TOTAL - 1);
```

</details>

<details>
<summary><b>🧲 Custom Magnetic Cursor</b></summary>

A two-part cursor (ring + dot) with lerp-based lag and magnetic snap on interactive elements. Uses CSS custom properties for smooth GPU-composited transforms.

</details>

<details>
<summary><b>💡 Light Bulb Toggle</b></summary>

A rope-pull light switch with SVG rope physics simulation. Click the rope to toggle between dark and light modes, complete with a rope-jiggle spring animation on pull.

</details>

<details>
<summary><b>🤖 Built-in AI Chat Assistant</b></summary>

An embedded AI assistant that answers questions about skills, projects, and availability — powered by a context-aware prompt injected at the portfolio level.

</details>

<details>
<summary><b>⌨️ Terminal Easter Egg + Konami Code</b></summary>

Type `help` in the floating terminal to discover hidden commands. Or try: `↑ ↑ ↓ ↓ ← → ← → B A` anywhere on the page.

</details>
